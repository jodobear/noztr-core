# Agent Prompt Template

## Part 1: Extract Boilerplate From A Mature Repo

```md
Review the current repo's process and extract:
- universal process rules
- project-profile rules
- project-local rules

Produce:
- process review
- process principles
- starter framework
- project profile template
- agent and phase prompt templates
- copyable starter files

Do not guess through high-impact ambiguity. Ask one targeted clarifying question when needed.
```

## Part 2: New Project Kickoff Prompt

```md
Start from these artifacts:
- `AGENTS.md`
- `docs/process/process-principles.md`
- `docs/process/project-profile.md`
- `docs/process/agent-prompt-template.md`
- `docs/process/phase-prompt-template.md`
- `docs/plans/decision-log.md`
- `docs/plans/build-plan.md`

Before planning or coding, confirm:
- purpose
- scope
- references
- toolchain
- dependency policy
- quality gates
- remote/push expectations

If any missing item materially changes setup, stop and ask one clarifying question.
```
