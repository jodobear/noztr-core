# noztr-core 0.1.0-rc.5

Date: 2026-03-24
Release type: breaking rc

## Summary

`0.1.0-rc.5` is the fifth public release candidate for `noztr-core`.

This RC removes `NIP-26` from the supported `noztr-core` surface. The protocol-kernel boundary,
ownership posture, and the rest of the public API stay otherwise unchanged.

## Public Highlights

- removed the public `nip26_delegation` route
- removed the `NIP-26` example and public support claims
- added an explicit downstream migration guide for the removal

## Breaking Changes

- `noztr-core` no longer exports `noztr.nip26_delegation`
- `NIP-26` is no longer listed as implemented in public coverage/reference docs
- migration guide:
  - [../guides/migrating-from-0.1.0-rc.4.md](../guides/migrating-from-0.1.0-rc.4.md)

## Compatibility Notes

- Zig toolchain floor for this RC line remains `0.15.2`
- optional I6 exports remain build-flag gated
- no other protocol routes were widened as part of this RC

## Verification

- `zig build lint`
- `zig build test --summary all`
- `zig build`

## Upgrade Guidance

- if you depend on `noztr-core`, review [../guides/migrating-from-0.1.0-rc.4.md](../guides/migrating-from-0.1.0-rc.4.md)
- remove any direct `noztr.nip26_delegation` usage
- refresh local symbol indexes or LLM context packs so they stop teaching the removed route
